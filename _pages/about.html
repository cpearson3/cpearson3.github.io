About me

I am a web and database application developer with 5+ years of professional experience, based in Morgantown, WV, USA. I have worked with numerous clients, from small businesses to universities, to design, develop, and support many types of applications. I always prioritize communication with the users and clients to establish a clear set of requirements. I'm strongly focused on delivering maintainable code and I pay attention to details.

The type of work I've done includes: Building database queries and stored procedures in PL SQL; developing reports with Oracle Reports, Excel, and the web; creating user interfaces for desktop database users with Oracle Forms Builder and for mobile web users using Angular, jQuery, Bootstrap and custom CSS frameworks; building Python and Shell scripts for system jobs and development tools; and managing projects with Git.
